Prerequisites for Using HyperIntelligence

The following prerequisites must be met to use HyperIntelligence.

MicroStrategy HyperIntelligence for Mobile and HyperIntelligence for Office do not currently support asymmetric clustering where different projects are loaded in each node of a MicroStrategy Intelligence Server cluster.

For more information on asymmetric clustering, see MicroStrategy Intelligence Server Clustering FAQ.

Necessary Privileges

To create HyperIntelligence Cards, you must have the Use Workstation privilege.

To certify cards, you need the Certify Content privilege. You must certify cards to deploy them.
