How to View Application Activity

You can view statistics and trends related to application activity. You must have permission to access the application through a connected environment.

You must configure Platform Analytics to view dossier usage activity.

The Platform Analytics cube must be published for the data to show up correctly and in real time.

The current user must have view access to the PA project.

To View Application Usage Activity

  1. Open the [%=Company_ProductNames.VizTool%] window.
  2. In the Navigation pane, click Applications.
  3. Right-click the application whose application activity you want to view and choose Properties.

  4. In the left pane, click Activity.

  5. View statistics and trends about the number of documents, dossiers, reports, and cubes in your application, as well as weekly and 30-day trends. You can also view a list of the top 25 objects, by total number of runs, for the last 30 days.
  6. Click OK.
