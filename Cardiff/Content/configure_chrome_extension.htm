Configure HyperIntelligence for Web

  1. Open the HyperIntelligence for Web extension in your browser.
  2. Enter an environment URL.

    Ensure the URL contains MicroStrategyLibrary at the end, like http://…/MicroStrategyLibrary

    How do I find my environment URL?
    1. From your cloud environment's welcome email, select Access MicroStrategy Platform.
    2. Hover over MicroStrategy Library and click Configure.
    3. Click Library Server.

    4. Under Security Settings, select All next to Allow Library embedding in other sites. This is necessary for cards to appear.

    5. Copy the URL under Library Server.
    6. Restart the Server.
  3. Click Continue.

  4. Log into your environment.

    What authentication modes are available for the extension?

The MicroStrategy logo for the HyperIntelligence extension will switch from gray to red.

How to Change the Environment URL

  1. Open HyperIntelligence for Web.
  2. Click on Settings.
  3. Click Edit on the settings page.
  4. Click Yes.
  5. Enter an environment URL.
  6. Click Save.

Once you have successfully logged into the environment, you will have access to any of the cards created in that environment.
