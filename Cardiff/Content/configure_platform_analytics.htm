How to Configure Platform Analytics

You must configure Platform Analytics before you can view dossier or application usage activity.

To Configure Platform Analytics

  1. Open the [%=Company_ProductNames.VizTool%] window with the Navigation pane in smart mode.
  2. In the Navigation pane, click Environments.
  3. You must connect to the environment that contains dossiers for which you want to view usage activity. Right-click the environment and choose Connect.
  4. Select the projects that contain the dossiers you want to use.
  5. Click OK.
  6. Right-click the environment again and choose Properties.
  7. In the Platform Analytics Environment drop-down, select the environment that contains your Platform Analytics project.
  8. Click OK.
