Install HyperIntelligence for Web

To use the HyperIntelligence for Web Chrome extension, either download the extension directly from the Chrome store or distribute the extensions via silent install.

This extension supports Chrome 69 and above on both Windows and Mac.

All users attempting to use HyperIntelligence for Web in MicroStrategy 2019 Update 2 must be assigned the Use Hyper Web privilege, directly or as part of a User Group or Security Role. This privilege can be found in this Client-Hyper-Web privilege group.

  • User
  • Admin

How to Add the HyperIntelligence Extension from the Chrome Store

  1. Open the Chrome Web Store to download the extension.
  2. Click Add to Chrome > Add Extension.

The MicroStrategy logo appears in your extension toolbar. Learn How to Configure the HyperIntelligence Chrome Extension.

How to Uninstall HyperIntelligence for Web

  1. Open the Chrome browser.
  2. At the top right, click More > More tools > Extensions.
  3. Locate HyperIntelligence, click Remove.
  4. Confirm by clicking Remove.

How to Add the HyperIntelligence Extension by Silent Install

As an admin, you may want to distribute the extension via silent install. Perform the following steps to silently install the extension on your user's browsers.

You can only add the extension by silent install on Windows.

Once installed via silent install, users cannot remove the extension through Chrome. See How to Uninstall the Chrome Extension for more information.

  1. Download HyperIntelligence for Web from the download site.

  2. Create a response file with the following parameters:

    [Installer]
HideAllDialogs=FALSE
[Welcome]
RemoveAll=FALSE
[Config]
TargetDir=[directory where the extension will be installed] EnvironmentUrl=[MicroStrategy Library server URL]

    Where

    TargetDir is the directory where the HyperIntelligence Chrome extension files will be stored. It's recommended to use the default directory C:\Program Files\MicroStrategy\HyperIntelligence.

    EnvironmentUrl is the MicroStrategy Library server URL where cards will be created.

  3. Save the file as response.ini.

  4. Close the Chrome browser and run the following command prompt:

    HyperIntelligenceSetup.exe --Responsefile="[response file path]" -s -f1"[path for log]"
  5. If you executed the command manually, click Yes on the message asking if HyperIntelligence can make changes to your device.

Users can now log in to the extension using their environment credentials. Learn how to Use HyperIntelligence for Web.

How to Uninstall HyperIntelligence for Web

When the extension is installed by silent install, users cannot remove the extension through the Chrome Store or the Chrome extension settings page. If a user manually removes the extension in either of these manners, the extension is placed on a Chrome blocklist. When this occurs, the extension cannot be updated or reinstalled using the installer.

How do I remove HyperIntelligence from the Chrome blocklist?

To remove HyperIntelligence from the blocklist, re-downloaded the extension through the Chrome Store and use the method below to uninstall the extension.

To correctly remove the extension, you must remove HyperIntelligence from your machine. Use the following steps to correctly uninstall HyperIntelligence for Web that was installed via silent install:

  1. Close the Chrome browser.
  2. On your Windows machine, go to Settings > Apps & features.
  3. Locate MicroStrategy HyperIntelligence and click Uninstall.
  4. Open Chrome and see the HyperIntelligence icon is no longer in the toolbar.
