Before creating cards, you want to ensure your dataset contains all necessary information, supported attribute relationships, and multi-form attributes that allow for alternate keyword matching.

Ensure your dataset is an in-memory cube. Live-connect is not supported. For example, you can create a card based on an Intelligent cube, super cube, or an in-memory MDX/SAP cube. You cannot create a card based on a live-connect MDX/SAP cube.
