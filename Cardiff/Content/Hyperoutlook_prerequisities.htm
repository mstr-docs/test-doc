Prerequisites for HyperIntelligence for Office

The following prerequisites must be met to use HyperIntelligence for Office.

System Requirements

  • MicroStrategy Library 2019 Update 2 server
  • MicroStrategy Workstation 2019 Update 2 to generate the manifest file. See Generate the Manifest File Using Workstation for more information.
  • MicroStrategy 2019+ Intelligence server

  • MicroStrategy 2019 Update 2 environment with an HTTPS domain

Supported Applications

  • Outlook for Mac (connected to Office 365)
  • Outlook on Windows (connected to Office 365)
  • Outlook Online (connected to Office 365)

For additional information, see Client requirements.

Supported API Requirement Sets

HyperIntelligence for Office requires the Outlook client to support Outlook JavaScript API v1.5 and above. For a list of supported API Requirements Sets, see API Requirements Sets.

Supported Operating Systems

  • Windows 10+
  • macOS High Sierra or above

Privileges

All users attempting to use HyperIntelligence for Office must be assigned the Use Hyper Office privilege, directly or as part of a User Group or Security Role. This privilege can be found in this Client-Hyper-Office privilege group.

Supported Web Browsers

Google Chrome is a certified PC browser.

The latest versions of the following PC browsers are supported:

  • Internet Explorer 11
  • Firefox
