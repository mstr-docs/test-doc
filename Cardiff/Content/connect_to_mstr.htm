Log in to HyperIntelligence for Office

The HyperIntelligence for Office add-in connects to the environment that generated the manifest file. If you need to connect to another environment, you need another add-in/manifest file with the new environment's information.

Since the manifest file has been deployed to user's accounts, they can now view cards by logging into HyperIntelligence on Outlook for desktop and Outlook for web.

Log into Outlook for Desktop

  1. Open Microsoft Outlook.
  2. Click the HyperIntelligence icon in the ribbon bar.
  3. Enter your environment's log in and password.

  4. Click the pin icon and click Log in with credentials.

Log into Outlook for Web

  1. Open Outlook in a supported web browser.

    Which web browsers are supported?

    Google Chrome is a certified PC browser. Firefox and Internet Explorer are also supported.
  2. Click an email and select the More actions icon.

  3. Click the HyperIntelligence icon.

  4. Click the pin icon and click Log in with credentials.
