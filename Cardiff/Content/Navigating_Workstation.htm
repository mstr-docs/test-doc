Navigate the Window

The window appears when you open . To open a new window, choose File > New Window. Use the window to navigate the system.

  1. Toggle between the icon and list view of your objects.
  2. Change how objects are arranged and sorted.
  3. Enter keywords to search for objects across all applications and environments.
  4. The Navigation pane is grouped into smart folders for your environments, dossiers, and more. Use it to access important resources such as Getting Started and MicroStrategy Community.
  5. Toggle the left Navigation pane between smart mode and folder mode. Smart mode, shown here, organizes your objects by type, across all environments and applications, and allows you to browse using the tabs. Folder mode allows you to manage applications and objects in a hierarchical view.
